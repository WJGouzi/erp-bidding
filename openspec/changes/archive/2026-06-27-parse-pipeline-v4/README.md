# parse-pipeline-v4: 专家思维三层解析 + 零LLM基础解析 + 编码修复 + PDF兼容

## 概述

基于 parse-pipeline-v3 的实战反馈，本次改造以"标书撰写专家的思维"重构解析管线，核心变化：

1. **零LLM基础解析** — Phase 1/2/3 的基础提取全部改为规则驱动，LLM仅保留用于Phase 3的策略分析
2. **三层专家架构** — 预设清单扫生死线 → 逐章节拆解评分点 → 横向关联定策略
3. **分包感知** — 每个分包独立处理生死线和评分
4. **编码全链路修复** — 确保中文不入库不乱码
5. **PDF兼容** — DOCX/PDF统一结构化模型
6. **表格解析增强** — 纯文本表格启发式检测

## 架构

```
┌──────────────────────────────────────────────┐
│          第1层：预设清单扫描（纯规则）            │
│  → 按 bid_type 加载模板                        │
│  → 关键词匹配 + 章节定位                       │
│  → 输出：资格门槛清单 + 废标条件 + ★条款         │
├──────────────────────────────────────────────┤
│          第2层：章节逐项拆解（结构化解构）         │
│  → 按文档章节顺序遍历                           │
│  → 表格优先解析评分表                           │
│  → 文本段落匹配得分点规则                        │
│  → 输出：评分得分点拆解表 + 撰写任务清单          │
├──────────────────────────────────────────────┤
│          第3层：横向关联分析（可选LLM）            │
│  → 跨包/跨章节整合                              │
│  → 竞争策略生成                                 │
│  → 输出：策略分析 + 资源配置优先级                │
└──────────────────────────────────────────────┘
```
